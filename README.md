# FAME Smart Flasher

This application is for flashing initial firmware for smart devices produced by Fyrby Additive during a build process, or for recovery purposes.

Devices already flashed with their own apps may have an easier update mechanism.

## Device Support

Currently this application supports the following smart devices:

* FAME Smart Blinds

## Platform Support

This application is available for the following platforms pre-built in the releases area:

* Windows (soon)
* macOS
* Android
* Linux Flatpak
